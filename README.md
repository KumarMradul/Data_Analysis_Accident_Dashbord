# Data_Analysis_Accident_Dashbord

🚦 Road Accident Analysis Dashboard (2021–2022)
📌 Project Overview

This project focuses on analyzing road accident data for the years 2021 and 2022 using an interactive dashboard. The goal is to provide meaningful insights into accident patterns, casualties, and trends to support better decision-making and road safety improvements.

🎯 Objectives
Analyze total casualties and total accidents
Track Year-over-Year (YoY) growth
Understand accident severity distribution
Identify trends based on vehicle type, location, and road type
Compare current year vs previous year performance
📊 Dashboard Preview




📈 Key KPIs
🔹 Primary KPIs
Total Casualties (Current Year vs Previous Year)
Total Accidents (Current Year vs Previous Year)
YoY Growth Analysis
🔹 Secondary KPIs
Total Casualties by Vehicle Type (Current Year)
📉 Analysis Included
📅 Monthly trend comparison (Current Year vs Previous Year)
🚗 Casualties by Vehicle Type
🛣 Casualties by Road Type
🌍 Casualties by Area/Location (Urban vs Rural)
🌙 Casualties by Day vs Night
📍 Total Accidents and Casualties by Location
🛠 Tools & Technologies Used
Power BI (Dashboard Creation)
Microsoft Excel / CSV (Data Source)
DAX (Data Analysis Expressions)


🔍 Key Insights
Significant variation observed in casualties between 2021 and 2022
Urban areas reported higher accident frequency compared to rural areas
Certain vehicle types contribute more to total casualties
Peak accidents occur during specific months and time periods
Road type plays a crucial role in accident severity
🚀 How to Use
Download the .pbix file from this repository
Open it using Power BI Desktop
Interact with filters (Year, Location, Road Type, etc.)
Explore insights through visualizations
📌 Conclusion

This dashboard provides a comprehensive view of road accident data, helping stakeholders identify patterns, improve safety measures, and make data-driven decisions.
